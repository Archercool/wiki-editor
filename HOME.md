# 欢迎使用知识库

基于 GitHub Pages 的在线可编辑知识库。

---

## 功能介绍

- **在线阅读**：点击左侧导航浏览文章
- **在线编辑**：点击右下角 ✏️ 按钮编辑当前文章
- **图片上传**：支持粘贴、拖拽、点击上传

---

## 快速开始

1. 点击左侧导航打开文章
2. 点击右下角 ✏️ 按钮进入编辑
3. 点击「GitHub 登录」完成认证
4. 编辑并保存
