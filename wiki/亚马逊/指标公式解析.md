## 2. 曝光&流量链路指标
衡量广告曝光、触达、点击吸引力
| 指标全称 | 简称 | 计算公式 | 业务释义 |
| ---- | ---- | ---- | ---- |
| Impressions | 曝光量 | 原始统计数值 | 广告整体展示总次数 |
| Viewable impressions | 有效可视曝光 | 原始统计数值 | 满足行业可视标准的曝光（画面可见≥1s） |
| User reach | 独立触达用户 | 原始统计数值 | 广告覆盖的去重买家数量 |
| Clicks | 点击量 | 原始统计数值 | 广告产生的总点击次数 |
| CTR | 整体点击率 | $$CTR = \frac{Clicks}{Impressions} \times 100\%$$ | 全部曝光下的点击转化比例，反映广告吸引力 |
| Viewable CTR (vCTR) | 可视点击率 | $$vCTR = \frac{Clicks}{Viewable\ Impressions} \times 100\%$$ | 仅统计有效可视曝光的点击，数据更真实 |
| Top-of-search impression share (IS) | 搜索首位曝光占比 | $$首位曝光占比 = \frac{自身搜索顶部曝光量}{市场顶部曝光总机会} \times 100\%$$ | 品牌抢占搜索首页顶部广告位的流量份额 |
| Top-of-search bid adjustment | 搜索顶部竞价溢价 | 手动设置固定数值 | 针对搜索首位位置设置的竞价上浮比例 |

## 3. 花费&成本计费指标
核算广告投入、单次曝光/点击成本
| 指标全称 | 简称 | 计算公式 | 业务释义 |
| ---- | ---- | ---- | ---- |
| Total cost | 总广告花费（Spend） | 原始统计数值 | 广告累计扣除的总费用 |
| CPC | 单次点击成本 | $$CPC = \frac{Total\ Cost}{Clicks}$$ | 平均每一次广告点击消耗的广告费 |
| CPM | 千次曝光成本 | $$CPM = \frac{Total\ Cost}{Impressions} \times 1000$$ | 每1000次广告曝光的平均花费（展示/视频广告核心） |
| Viewable CPM (vCPM) | 千次可视曝光成本 | $$vCPM = \frac{Total\ Cost}{Viewable\ Impressions} \times 1000$$ | 每1000次有效可视曝光的平均花费 |

## 5. 核心转化&投产指标
广告转化订单、销售额、盈利效率核心指标
| 指标全称 | 简称 | 计算公式 | 业务释义 |
| ---- | ---- | ---- | ---- |
| Purchases | 广告总订单 | 原始统计数值 | 广告归因周期内全部成交订单 |
| Sales | 广告总销售额 | 原始统计数值 | 广告订单对应的成交总金额 |
| Advertising cost of sales | ACOS广告销售成本 | $$ACOS = \frac{Total\ Cost}{Sales} \times 100\%$$ | 广告花费占广告销售额的百分比，越低越盈利 |
| ROAS | 广告投入产出比 | $$ROAS = \frac{Sales}{Total\ Cost}$$ | 每投入1单位广告费，带来的销售金额 |

### 🔁 ACOS & ROAS 换算关系
$$ROAS = \frac{1}{ACOS} \quad , \quad ACOS = \frac{1}{ROAS} \times 100\%$$
> 示例：ACOS=20% → ROAS=5；ROAS=4 → ACOS=25%

## 6. 新客（New-to-Brand）拓展指标
衡量广告拉新能力，区分老客/首次购买品牌用户
| 指标名称 | 计算公式 |
| ---- | ---- |
| Purchases (new to brand) | 新客订单数（原始统计值） |
| Sales (new to brand) | 新客销售额（原始统计值） |
| Percent of purchases new to brand | $$\frac{新客订单数}{广告总订单数} \times 100\%$$ |
| Percent of sales new to brand | $$\frac{新客销售额}{广告总销售额} \times 100\%$$ |

## 8. 视频广告播放效果指标
### 8.1 原生计数指标（无基础公式）
视频播放各阶段触发次数，系统直接统计：
- First quartile (video ad)：视频播放至25%
- Midpoint (video ad)：视频播放至50%
- Third quartile (video ad)：视频播放至75%
- Complete views (video ad)：视频完整播放完毕
- Unmutes (video ad)：用户手动开启视频声音次数
- Viewable distribution：可视曝光分层分布（展示型广告分层数据）

### 8.2 衍生播放完成率公式
| 衍生指标 | 计算公式 |
| ---- | ---- |
| 视频完整播放率 VCR | $$VCR = \frac{Complete\ views}{Viewable\ Impressions} \times 100\%$$ |
| 25%播放完成率 | $$Q25播放率 = \frac{First\ quartile}{Viewable\ Impressions} \times 100\%$$ |
| 50%播放完成率 | $$Q50播放率 = \frac{Midpoint}{Viewable\ Impressions} \times 100\%$$ |
| 75%播放完成率 | $$Q75播放率 = \frac{Third\ quartile}{Viewable\ Impressions} \times 100\%$$ |
